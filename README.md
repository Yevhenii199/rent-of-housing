## 📄 Overview
This project is a simple adaptive landing page for renting a guest house.
The site is adapted for Russian-speaking, English-speaking and Serbian-speaking users.

## 🎨 Design Features
- **Responsive grid layout** using CSS Grid for room cards
- **Image carousels** for each room
- **Mobile-friendly**: media queries adapt the layout for all screen sizes
- **Modern typography**: Montserrat font from Google Fonts
- **Contact buttons** linking to:
  - Facebook Messenger
  - Telegram
  - Viber
  - Gmail

## 🧩 Technologies
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Google Fonts
- WebP image format
- JavaScript (optional, used for the image carousel)

## 🔧 How to Run Locally
1.Clone the repository:

```bash
git clone https://github.com/Yevhenii199/rent-of-housing 
```

2.Navigate to the project directory:

```bash
cd rent-of-housing
```

3.Open the **index.html** file in your browser

```
rent-of-housing/
│
├── images/         # Folder with images of the house and rooms
├── index.html      # Home page in Russian
├── en.html         # English version of the site
├── sr.html         # Serbian version of the site
├── styles.css      # General style file
│            
└── README.md       # Project information (you are reading this file)
```
