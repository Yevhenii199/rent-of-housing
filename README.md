## rent-of-housing
This project is a simple adaptive landing page for renting a guest house.
The site is adapted for Russian-speaking, English-speaking and Serbian-speaking users.

The site contains:

- Information about the guest house and its location
- Photos of rooms, living room, kitchen and other areas
- Description of amenities and services
- Contact links for communication via Facebook, Telegram, Viber and Email
- Built-in Google Maps with the exact location of the house

```
rent-of-housing/
│
├── images/         # Folder with images of the house and rooms
├── index.html      # Home page in Russian
├── en.html         # English version of the site
├── sr.html         # Serbian version of the site
├── styles.css      # General style file
│            
└── README.md       # Project information (you are reading this file)
```
🚀 Getting Started
To run this project locally:

Clone the repository: git clone https://github.com/Yevhenii199/rent-of-housing 
Open index.html directly in your browser.
No build tools or frameworks are required for this version — it's built with plain HTML and CSS.
